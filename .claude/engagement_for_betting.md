# 馬券購入促進機能 - 金沢競馬を応援する

## プロジェクトの使命

**金沢競馬場が廃止にならないように、ユーザーに馬券を買いたくなる情報を提供し、実際に購入を促す**

**重要**: 全国どこからでも楽天競馬などのオンラインサービスを通じて金沢競馬の馬券を購入できます。場外・ネット投票の売上も金沢競馬場の存続に貢献します。

## 基本方針

1. **少額でも楽しい**: 100円からでも参加できる楽しさを訴求
2. **応援する喜び**: 馬券購入が「金沢競馬への応援」だと実感させる
3. **発見と驚き**: データから面白い情報を見つけ、興味を引く
4. **購入体験を簡便に**: 楽天競馬へのスムーズな導線
5. **ソーシャル要素**: みんなで応援する一体感
6. **⚠️ 予想の押し付けは禁止**: AI予想の3連単を「買って」とは言わない。あくまで「参考情報」として提示

## 機能一覧

### 1. 「今週のイチオシ」セクション

**目的**: 毎週の注目レースと馬を紹介し、購入動機を作る

**表示内容**:
- **今週の注目レース**: メインレースやG3など
- **今週の注目馬**: 好調な馬、連勝中の馬、復活の兆し
- **今週の注目騎手**: 好成績の騎手
- **プチストーリー**: 「この馬は3連勝中！」「ベテラン〇〇歳が復活！」

**配置**: トップページの目立つ位置

**更新頻度**: 毎週月曜日

### 2. 「100円から応援」訴求

**目的**: 少額購入のハードルを下げる

**表示内容**:
- 「100円の馬券で金沢競馬を応援！」
- 「コーヒー1杯分で夢を買おう」
- 「少額でも当たれば数千円に！」

**具体例**:
```
100円の3連単が当たれば...
平均配当: 3,420円（34倍）
最高配当: 125,840円（1,258倍）

たった100円で金沢競馬を支えられます！
```

**配置**: 予想ページのサイドバー、フッター

### 3. 「みんなの購入状況」表示

**目的**: ソーシャルプルーフで購入を後押し

**表示内容**:
- 「今週〇〇人が馬券を購入しました！」
- 「このレースは〇〇人が注目」
- 「この馬は〇〇人がお気に入り登録」

**データソース**:
- 楽天競馬のAPI（もしあれば）
- 自サイトのお気に入り登録数
- 予想ページの閲覧数

**配置**: レース詳細ページ、予想ページ

### 4. 購入導線の最適化

**目的**: 楽天競馬への移動をスムーズに

**実装内容**:
- **「楽天競馬で買う」ボタン**: 目立つ配置、各レースページに設置
- **ワンタップ購入**: 予想の3連単をそのまま楽天競馬で購入できるリンク
- **QRコード**: スマホで簡単アクセス
- **アフィリエイト**: 楽天競馬のアフィリエイトプログラム活用（収益化も）

**ボタンデザイン例**:
```
┌─────────────────────────┐
│ 🎯 楽天競馬で馬券を買う │
│   (100円から購入可能)    │
└─────────────────────────┘
```

### 5. 当たった人の声（実績紹介）

**目的**: 「自分も当たるかも」という期待感を醸成

**表示内容**:
- 「先週の的中報告: 〇〇さんが3連単的中！」
- 「少額購入で○万円ゲット！」
- 「初めての馬券で当たりました！」

**実装方法**:
- ユーザーが自己申告で報告（任意）
- Twitter連携で的中ツイートを自動収集
- 匿名で「〇〇県の方」として紹介

**配置**: トップページ、購入完了ページ

### 6. ゲーミフィケーション

**目的**: 継続的な購入を楽しくする

**実装内容**:
- **バッジ制度**:
  - 「初購入バッジ」
  - 「10回購入バッジ」
  - 「的中バッジ」
  - 「金沢競馬サポーターバッジ」
- **ランキング**:
  - 的中回数ランキング
  - 購入回数ランキング（応援度ランキング）
- **チャレンジ**:
  - 「今月5回購入で特別バッジ」
  - 「3連続購入で〇〇プレゼント」

**報酬**:
- デジタルバッジ
- サイト内での称号
- 限定コンテンツへのアクセス権

### 7. 「金沢競馬を守る」意識の醸成

**目的**: 馬券購入が社会貢献だと実感させる

**表示内容**:
- **売上貢献度の可視化**:
  - 「あなたの購入で金沢競馬が〇〇円支えられました」
  - 「今月の総売上: 〇〇億円（前年比+5%）」
- **競馬場存続の危機感**:
  - 「地方競馬の現状: 全国で〇〇場が廃止」
  - 「金沢競馬を未来に残すために」
- **感謝メッセージ**:
  - 「ご購入ありがとうございます！金沢競馬を応援してくださり感謝です」

**配置**: 購入完了ページ、統計ページ

### 8. データから「買いたくなる」情報を抽出

**目的**: 統計や分析から購入意欲を刺激

**表示内容**:
- **「今が買い時」の馬**:
  - 「3連勝中の〇〇、今週も注目」
  - 「人気薄だけど実力馬、〇〇が狙い目」
  - 「ベテラン騎手×若手馬の好相性」
- **高配当の可能性**:
  - 「この組み合わせなら配当10万円超えも！」
  - 「穴馬候補: 人気薄だが過去に好走」
- **期待値分析**:
  - 「この予想の期待値は〇〇円（購入推奨）」
  - 「オッズから見た狙い目馬」

### 9. 購入体験の記録・共有

**目的**: 購入後の満足度を高め、リピートを促す

**実装内容**:
- **マイページ**:
  - 購入履歴の一覧
  - 的中率・回収率の表示
  - 総購入額・総払戻額の集計
- **購入日記**:
  - 「なぜこの馬を選んだか」メモ機能
  - レース結果と感想の記録
- **SNS共有**:
  - 「今日は〇〇に賭けました！」とTwitter投稿
  - 的中時の自動シェア機能

### 10. メール・プッシュ通知

**目的**: タイムリーに購入を促す

**通知内容**:
- **レース前日**: 「明日は金沢競馬！今週のイチオシは〇〇」
- **レース当日**: 「本日14時から開催！予想をチェック」
- **締切30分前**: 「まもなく締切！最後のチャンス」
- **お気に入り馬の出走**: 「あなたのお気に入り〇〇が出走します」

**配信タイミング**: ユーザーが設定可能

### 11. 初心者向けガイド

**目的**: 馬券購入未経験者のハードルを下げる

**コンテンツ**:
- **「はじめての馬券購入」ガイド**:
  - 楽天競馬への登録方法
  - 馬券の買い方（画面付き解説）
  - 3連単とは？単勝とは？
  - 少額購入のススメ
- **FAQ**:
  - 「いくらから買える？」→ 100円から
  - 「当たる確率は？」→ 3連単は難しいが当たれば高配当
  - 「どうやって買う？」→ 楽天競馬で簡単
- **動画解説**: YouTube等で購入手順を紹介

**配置**: ヘッダーメニュー、フッター

### 12. 「応援購入」コンセプト

**目的**: 馬券を「ギャンブル」ではなく「応援」と位置づける

**メッセージ例**:
- 「馬券購入は金沢競馬への応援チケット」
- 「当たらなくても、あなたの100円が馬と騎手を支えます」
- 「金沢の文化を未来に残すために」

**ビジュアル**:
- 競馬場の写真、馬の写真

- 応援メッセージの吹き出し

### 13. リマインダー機能

**目的**: 購入忘れを防ぐ

**実装内容**:
- **カレンダー登録**: 開催日をGoogleカレンダーに追加

- **定期購入の提案**: 「毎週日曜日に通知」設定

### 14. 限定キャンペーン訴求

**目的**: 楽天競馬のキャンペーンを紹介し購入を促す

**表示内容**:
- 「今なら楽天ポイント2倍！」
- 「新規登録で1,000円分のポイントプレゼント」
- 「的中で楽天ポイント還元」

**配置**: バナー、ポップアップ

## UI/UXの工夫

### デザイン方針
- **購入ボタンは目立つ色**: オレンジ、赤などアクション色
- **ポジティブな言葉**: 「応援」「支える」「一緒に」
- **感情に訴える**: 馬の写真、騎手の写真、競馬場の風景
- **シンプルな導線**: 3クリック以内で購入ページへ

### モバイル最適化
- スマホでタップしやすい大きなボタン
- 縦スクロールで情報が完結
- 購入ボタンは常に画面下部に固定

## 法的・倫理的配慮

### 注意事項の明記
- 「ギャンブル依存に注意」
- 「20歳未満は購入できません」
- 「余剰資金で楽しみましょう」
- 「必ず当たるわけではありません」

### 適正な表現
- ❌ 「絶対当たる」「必ず儲かる」「この予想を買って」
- ⭕ 「的中の可能性」「期待値が高い」「応援しよう」「参考情報としてご覧ください」

### ⚠️ 重要な制約
**AI予想の3連単について**:
- 「この予想を買ってください」とは言わない
- 「参考情報」「データ分析結果」として提示する
- ユーザー自身の判断を尊重する
- 「楽天競馬で馬券を買う」ボタンは、予想とは別の文脈で表示（「金沢競馬を応援する」という一般的な訴求）

**正しい表現例**:
- ⭕ 「AI予想は参考情報です。自己責任でご判断ください」
- ⭕ 「データから導き出した一つの見解です」
- ⭕ 「金沢競馬を楽しむ際の参考にしてください」
- ⭕ 「馬券購入は楽天競馬で。100円から金沢競馬を応援できます」

**避けるべき表現**:
- ❌ 「この3連単を買おう！」
- ❌ 「予想通りに買えば当たる」
- ❌ 「今すぐこの組み合わせを購入」

## KPI（成功指標）

### 追跡すべき指標
- **購入転換率**: サイト訪問者のうち何%が購入したか
- **リピート率**: 再度購入したユーザーの割合
- **平均購入額**: 1回あたりの購入金額
- **滞在時間**: 予想ページの閲覧時間
- **シェア数**: SNSでのシェア回数

### 目標設定
- 月間購入ユーザー数: 100人
- リピート率: 50%以上
- 平均購入額: 500円以上
