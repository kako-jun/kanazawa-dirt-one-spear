# プロジェクト概要

## 動機：100円馬券を自動的に買い続けたい

競馬の本質は「情報の非対称性を突く」ことだ。プロや集合知に押されて、個人が手計算で勝ち続けるのは難しい。では機械学習が相手ならどうか。AIが大量の過去データを統計的に処理し、人間の目には見えないパターンを発見できるなら、少額の馬券を継続的に買い続けることで、長期的にプラス収支が得られるかもしれない。

このプロジェクトはその「もしかしたら」を実際に試すための装置だ。1レースあたり100円。的中しなくても許容できる金額で、何百回も試し続ける。小さく賭けて、記録して、改善する。それを繰り返す。

## なぜ金沢競馬場か

中央競馬（JRA）は競合が多い。GPUファーム規模のAIが既に精度の高い予想を出しており、個人が入り込む余地は小さい。

地方競馬は違う。データが少なく、分析ツールも少ない。競合が少ない市場には、まだ拾える非効率が残っている。金沢競馬場は石川県金沢市に位置する公営の地方競馬で、ダートコースのみ。馬券はネット購入が可能で、個人が試しやすい環境が整っている。

## 地方競馬の特殊性

地方競馬には中央競馬にはない特性がある：

- **コースの固定性**：金沢はダートのみ。馬場状態の影響が中央より大きく出やすい
- **騎手の固定性**：所属騎手が限られ、騎手と馬のコンビが長期的に継続する傾向がある
- **少頭数レース**：1レースの出走頭数が少ない。3連単の組み合わせ数が減り、的中の難易度が下がる
- **出走間隔の短さ**：同じ馬が短い間隔で出走するため、最近の調子を把握しやすい
- **データの偏り**：公開されているデータ量が中央より少なく、モデルが過学習しにくい反面、精度向上にも限界がある

## コンセプト：一本槍

予想は3連単1点のみ。「もしかしたら当たるかもしれない多点買い」ではなく、1点だけに絞り込む。それが当たれば大きい。外れれば100円の授業料。どちらでも記録が積み重なり、システムの改善につながる。

「一本槍」という名前はここから来ている。潔く、清潔に、1点だけを指す。

## 楽しみ方

- AIが何を根拠に選んだかを確認できる（説明可能な予想）
- 毎回の予想と結果を記録し、長期的なパフォーマンスを見る
- オッズを使う予想と使わない予想を両立させ、どちらが賢いかを競わせる
- 試行錯誤のプロセスそのものをコンテンツにする

負けても「データが増えた」と思える設計にしてある。
